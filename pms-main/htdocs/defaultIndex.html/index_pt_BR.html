Oracle HTTP Server 12c  




O Oracle HTTP Server 12c se baseia na tecnologia do Servidor HTTP Apache de código-fonte aberto, cuja qualidade é comprovada, e fornece um framework para hospedar páginas Web dinâmicas e estáticas, além de funcionar como front-end para os Aplicativos do Oracle Fusion Middleware.






Funcionalidades


Servidor de Conteúdo/Proxy Reverso

Suporte a Implantação em Nuvem/Servidor Virtual
Centenas de sites/domínios de aplicativos atendidos por uma única instância do servidor Web. Cada servidor virtual pode ter seus próprios arquivos de configuração, endereços IP, portas, diretórios-raiz de documentos, preferências, arquivos de log e muito mais.

Proteção contra Ameaças Comuns
O módulo interno ModSecurity permite configurar regras para introspectar e proteger os aplicativos contra ataques comuns, incluindo injeção de SQL/Comando, Cross Site Scripting e outras vulnerabilidades.

Suporte a FastCGI
Uma forma eficiente de oferecer páginas Web de conteúdo dinâmico no OHS usando linguagens de script como PHP ou Python, sem perda significativa de desempenho.

Proxy Reverso Integrado
Módulos internos de proxy fornecem suporte genérico ao uso de proxies e suporte otimizado ao WebLogic Server, permitindo que o OHS atue como o ponto final de HTTP para servidores de origem HTTP, incluindo o WebLogic Server.

Administração/Monitoramento

Administração do Servidor
Utiliza as interfaces de administração do WebLogic 12c para fornecer um modelo de administração simples, consistente e distribuído que permite administrar o Oracle HTTP Server, o Oracle WebLogic Server e o restante da Pilha do Fusion Middleware.

Para obter mais informações, consulte a seção Conceitos Básicos do Modelo de Administração do OHS.

Monitoramento
A integração com o Oracle Enterprise Manager permite que os clientes monitorem o tráfego usando a console do Oracle Enterprise Manager.

Ferramenta de Migração Robusta
Ferramentas de migração integradas facilitam a migração de implantações existentes do Oracle HTTP Server 11g para o Oracle HTTP Server 12c.
