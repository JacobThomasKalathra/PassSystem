Oracle HTTP Server 12c  




Oracle HTTP Server 12c 是以經過實證的開放原始碼 Apache HTTP Server 技術為基礎，提供架構代管靜態、動態網頁及前端 Oracle Fusion Middleware 應用程式。






功能


提供內容服務 / 反向代理主機

支援雲端部署 / 虛擬伺服器
由單一 Web 伺服器執行處理為數以千計的網站 / 應用程式提供網域服務。每個虛擬伺服器都可以有自己的組態檔案、IP 位址、連接埠、文件根目錄、偏好設定、日誌檔及許多其他項目。

保護不受常見威脅的防護功能
內建的 ModSecurity 模組可供設定規則，進行自我檢查並保護應用程式不受常見攻擊的威脅，例如 SQL/命令插入攻擊、跨網站命令檔漏洞攻擊及其他漏洞攻擊。

FastCGI 支援
使用包括 PHP 或 Python 等的命令檔語言，有效率地在 OHS 中提供動態內容網頁服務，而不會造成明顯的效能損失。

整合式反向代理主機服務
內建的代理主機模組除了提供一般的代理主機支援外，還提供 WebLogic 伺服器的最佳化服務，可讓 OHS 作為 HTTP 原始伺服器 (包括 WebLogic 伺服器) 的 HTTP 端點。

管理 / 監督

伺服器管理
運用 WebLogic 12c 管理介面提供一個簡單、一致且分散式的管理模型，用於管理 Oracle HTTP Server、Oracle WebLogic Server 及其他 Fusion Middleware 堆疊。

如需詳細資訊，請參閱瞭解 OHS 管理模型一節。

監督
整合 Oracle Enterprise Manager 讓使用者可以使用 Oracle Enterprise Management 主控台監督 HTTP 流量。

健全的移轉工具
整合式移轉工具可以輕鬆地從現有的 Oracle HTTP Server 11g 部署移轉到 Oracle HTTP Server 12c。
