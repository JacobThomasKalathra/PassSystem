Oracle HTTP Server 12c  




Oracle HTTP Server 12c는 입증된 오픈 소스 Apache HTTP Server 기술을 기반으로 하며 정적, 동적 웹 페이지를 호스트하고 Oracle Fusion Middleware 응용 프로그램의 프론트 엔드로 사용하기 위한 프레임워크를 제공합니다.






기능


콘텐츠 제공/역방향 프록시

클라우드 배치/가상 서버 지원
수천 개의 사이트/응용 프로그램 도메인이 단일 웹 서버 인스턴스에서 처리됩니다. 각 가상 서버에 자체 구성 파일, IP 주소, 포트, 문서 루트, 환경설정, 로그 파일 등이 있을 수 있습니다.

일반적인 위협으로부터 보호
내장 ModSecurity 모듈이 응용 프로그램을 검사하고 SQL/명령 삽입, 사이트 간 스크립팅 취약점 및 기타 취약점을 비롯한 일반적인 공격으로부터 보호하는 규칙을 구성하는 기능을 제공합니다.

FastCGI 지원
성능에 큰 영향을 주지 않고 PHP 또는 Python과 같은 스크립팅 언어를 사용하여 OHS 내에서 동적 콘텐츠 웹 페이지를 제공하는 효율적인 방법입니다.

통합 역방향 프록시
내장 프록시 모듈이 일반 프록시 지원 및 WebLogic Server에 최적화된 지원을 제공하여 OHS가 WebLogic Server를 비롯한 HTTP 원천 서버의 HTTP 끝점으로 작동할 수 있게 합니다.

관리/모니터링

서버 관리
WebLogic 12c 관리 인터페이스를 사용하여 Oracle HTTP Server, Oracle WebLogic Server 및 Fusion Middleware 스택의 기타 제품을 관리하기 위한 간단하고 일관된 분산 관리 모델을 제공합니다.

자세한 내용은 Understanding the OHS Administration Model 단원을 참조하십시오.

모니터링
Oracle Enterprise Manager와 통합되어 고객이 Oracle Enterprise Management 콘솔을 사용하여 HTTP 트래픽을 모니터할 수 있게 합니다.

강력한 이전 툴
통합된 이전 툴을 사용하여 기존 Oracle HTTP Server 11g 배치를 Oracle HTTP Server 12c로 쉽게 이전할 수 있습니다.
