Oracle HTTP Server 12c  




Oracle HTTP Server 12c si basa su Apache HTTP Server, una tecnologia open source affidabile in grado di fornire una piattaforma avanzata per effettuare l'hosting di pagine Web statiche e dinamiche e il supporto front-end per le applicazioni Oracle Fusion Middleware.






Funzioni


Servizio contenuto/Proxy inverso

Distribuzione su cloud/Supporto di server virtuali
Migliaia di siti/domini applicazione da un'unica istanza del server Web. Ogni server virtuale può disporre di file di configurazione, indirizzi IP, porte, radici dei documenti, preferenze, file di log propri e molto altro ancora.

Protezione da minacce comuni
Il modulo ModSecurity integrato fornisce la capacità di configurare regole per salvaguardare e proteggere le applicazioni da attacchi comuni, tra cui l'inserimento di SQL/comandi, violazioni di tipo Cross Site Scripting (XSS) e altri tipi di violazioni.

Supporto di FastCGI
Metodo efficiente per servire pagine Web con contenuto dinamico all'interno di OHS mediante l'utilizzo di linguaggi di script, ad esempio PHP o Python, senza ripercussioni significative a livello di prestazioni.

Proxy inverso integrato
I moduli proxy integrati forniscono un supporto proxy generico, nonché un supporto ottimizzato per WebLogic Server, consentendo a OHS di fungere da endpoint HTTP per server di origine HTTP, incluso WebLogic Server.

Amministrazione/Monitoraggio

Amministrazione server
Si avvale delle interfacce di amministrazione di WebLogic 12c per fornire un modello di amministrazione semplificato, omogeneo e distribuito per l'amministrazione di Oracle HTTP Server, Oracle WebLogic Server e di tutte le altre applicazioni Fusion Middleware.

Per ulteriori informazioni, fare riferimento alla sezione Understanding the OHS Administration Model.

Monitoraggio
L'integrazione con Oracle Enterprise Manager consente agli utenti di monitorare il traffico HTTP mediante l'utilizzo di Oracle Enterprise Management Console.

Strumento di migrazione affidabile
Gli strumenti di migrazione integrati semplificano in modo significativo la migrazione delle distribuzioni esistenti di Oracle HTTP Server 11g a Oracle HTTP Server 12c.


Documentazione e collegamenti utili
