Oracle HTTP Server 12c  




Oracle HTTP Server 12c est basé sur la technologie de serveur HTTP Apache Open Source éprouvée. Il fournit une structure pour l'hébergement des pages Web dynamiques et statiques, ainsi que pour les applications frontales Oracle Fusion Middleware.






Fonctionnalités


Proxy inverse/service de contenu

Prise en charge de serveur virtuel/déploiement sur le cloud
Des milliers de domaines d'application/sites desservis à partir d'une instance de serveur Web unique. Chaque serveur virtuel peut avoir ses propres fichiers de configuration, adresses IP, port, répertoire racine de documents, préférences, fichiers journaux, etc.

Protection contre les menaces courantes
Le module ModSecurity intégré permet de configurer des règles pour inspecter les applications et les protéger contre les attaques courantes, notamment des vulnérabilités liées à l'injection de commandes/SQL, des vulnérabilités liées au scriptage inter-site et autres.

Prise en charge FastCGI
Moyen efficace de desservir des pages Web au contenu dynamique dans OHS à l'aide des langages de script comme PHP ou Python, sans prendre le risque d'altérer considérablement les performances.

Proxy inverse intégré
Les modules proxy intégrés fournissent une prise en charge de proxy générique, ainsi qu'une prise en charge optimisée de WebLogic Server, ce qui permet à OHS d'agir en tant qu'adresse HTTP pour les serveurs d'origine HTTP, y compris pour WebLogic Server.

Administration/Surveillance

Administration du serveur
Utilise les interfaces d'administration WebLogic 12c afin de fournir un modèle simple, cohérent et distribué pour l'administration d'Oracle HTTP Server, d'Oracle WebLogic Server et du reste de la pile Fusion Middleware.

Pour plus d'informations, reportez-vous à Description du modèle d'administration OHS.

Surveillance
L'intégration avec Oracle Enterprise Manager permet aux clients de surveiller le trafic HTTP à l'aide de la console Oracle Enterprise Management.

Outil de migration robuste
Les outils de migration intégrés permettent de migrer facilement les déploiements Oracle HTTP Server 11g vers Oracle HTTP Server 12c.


Documentation et liens utiles

La documentation du produit contient les guides d'installation et d'administration, ainsi que des notes sur la version.
